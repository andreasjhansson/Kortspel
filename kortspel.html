GitHub Gist-synk
Synka data mellan enheter via ett privat GitHub Gist.
Så här fungerar det:
1. Skapa ett GitHub Personal Access Token med gist-behörighet.
2. Klistra in token nedan och klicka Skapa Gist.
3. Kopiera Gist-ID och ange det på dina andra enheter.
Sparas bara lokalt i webbläsaren. Skapa token →
Ange samma Gist-ID på alla enheter som ska synka.
Din data lagras i ett privat Gist på ditt GitHub-konto. Ingen utomstående kan se den.
